# Event Management System
An Application where users can view and register for events while administrators can manage events.

---

## Tech Stack
- React + TypeScript
- Tailwind CSS (Styling)
- Formik (Form Handling)
- Yup (Form Validation)
- React Router (Navigation)
- Vite
- Vitest (Unit Testing)

---

## Getting Started
```bash
npm install
npm run dev
```

Application runs on:
```bash
http://localhost:5173/
```
